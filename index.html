AlphaLoom

文字列組み合わせ＆辞書照合による鍵候補探索ツール

1) 入力

文字候補は A–Z のみ（空欄可）。空欄の列は A–Z 全てを等確率で扱います。
文字を並べた順に重みが強くなります（例: CAB は C > A > B）。

2) 結果（列重みベースの信頼度）

高信頼
中信頼
小信頼

エクスポート

辞書設定

辞書：内蔵ミニ辞書（約200語）

辞書情報

合計単語数: 0 語

読み込み済み辞書

辞書ファイル仕様

  • 必須: 1行1ワード、すべて大文字（A-Z）
  • 推奨: ソート済み・重複除去済み（処理効率向上のため）
  • 文字エンコーディング: UTF-8

使い方

  • ローカルファイル: 上記仕様のテキストファイルを選択して「辞書を読み込み」をクリック
  • GitHub Pages: 相対パスを入力して「fetchで取得」をクリック
  • 正規化: 自動的に A–Z のみに変換し、大文字化・重複は除去

使い方（概要）

  1. 鍵文字数（1〜20）を選ぶと「列1〜L」の入力欄が生成されます。
  2. 各列に候補文字（A–Z, 空欄可）を入力します。並び順が重みになります。
  3. 「全パターン出力」で候補鍵を生成・スコア化し、高/中/小に自動分類します。
  4. 「辞書による絞り込み」で、鍵そのものの完全一致/部分一致を使って再ランクします。
  5. 必要に応じて辞書タブで辞書を追加・有効化してください。
